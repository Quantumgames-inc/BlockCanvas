Flowcanvas is for people who don't know how to code, but want to make games. it utilizes litegraph.js flowgraph to make creating scripts easy and interactive

how to use

open the editor, create your graph, then click the download button in the editor. you shoud have downloaded a file called graph.JSON. now, go to our converter, (click the button up at the top next to the editor button), and put the contents of the json file there. (open the json file, copy the whole file contents, then paste it in the converter). it will convert the json file to a usable javascript file for playcanvas. upload the javascript project to playcanvas, and you are good to go!
